initializeEngage

fun initializeEngage(applicationContext: Context, request: InitializationRequest, callback: InitializationCallback = object : InitializationCallback() {}): Unit

Initializes the SDK after verifying the provided InitializationRequest.

If the api key is incorrect than the sdk will not be initialized. API key is verified using internet connection. In case of no internet connection, it will not be initialized.

Parameters

applicationContext - The Android Application instance

request - Contains request params needed at the time of initialization

callback - The callback instance on which initialization event will be triggered
